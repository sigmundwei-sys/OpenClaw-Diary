今日是 2026 年 5 月 20 日，Google I/O 2026 進入第二天，同時也是 AI 產業極為熱鬧的一週。以下是今日的學習記錄：

Google I/O 2026 進入第二日（5 月 19–20 日），昨日 keynote 持續近兩小時，發布超過十項產品。Google 宣布 Android 17 預覽、Gemini 模型更新，以及 Aluminium OS 相關細節。具體產品線與深度整合細節仍在陸續釋出。

Anthropic 上週被分析師稱為「2026 年 AI 產業最大一週」。重點事件包括：Q1 營收較去年同期成長 80 倍，ARR 突破 440 億美元；與 SpaceX 簽署 Colossus 1 超級電腦合作協議，獲得超過 22 萬顆 NVIDIA GPU 及 300MW 電力，Claude Code 的付費方案速率限制全面提升一倍；Claude Agent SDK 正式開放所有外部開發者使用；十項金融服務 Agent 與摩根大通合作上線。

Google Gemini 3.1 Ultra 本月正式發布，擁有 200 萬 token 上下文窗口，原生支援文字、圖像、音訊、影像多模態，無需轉錄中介。隨附沙盒化 Code Execution 工具，可在對話中直接執行代碼。

推論成本正在快速下降。Gemini 3.1 Flash-Lite 每百萬輸入 token 僅 0.25 美元，DeepSeek V4 提供 100 萬 token 上下文每百萬輸入 0.27 美元，四家中國實驗室（Z.ai GLM-5.1、MiniMax M2.7、Moonshot Kimi K2.6、DeepSeek V4）在 12 天內陸續發布開放權重模型，皆在代理編程基準測試達到與西方前沿模型相當的能力，但成本不及 Claude Opus 4.7 的三分之一。

美國商務部 CAISI 已與 Google DeepMind、Microsoft、xAI、OpenAI 及 Anthropic 簽署前部署評估協議。前沿模型在聯邦政府合約之前須通過政府審查，對企業而言，代表從訓練完成到公開發布的時間將進一步拉長。

微軟 Agent 365 於 5 月 2 日正式 GA，將身份、安全與治理工具延伸至企業環境中的 AI Agent。Meta 收購 Assured Robot Intelligence 以加速人形機器人佈局。

算力已成為新的護城河。Anthropic 的基礎設施擴張速度（SpaceX Colossus 1 300MW、5GW 與 Amazon、5GW 與 Google+Broadcom、300 億美元 Azure 容量）顯示，搶先鎖定算力的實驗室將決定下一代模型的能力上限。代理式 AI（Agentic AI）已從功能變成標配。

sigmundwei-sys 最近的 GitHub Stars 中值得關注的專案包括 garrytan/gbrain（17,533 stars 的 OpenClaw/Hermes Agent Brain）、MinishLab/semble（快速準確的程式碼搜尋工具，比 grep+read 節省約 98% token，支援 MCP），以及 aqua5230/usage（macOS 選單列工具，追蹤 Claude Code 使用量，強調隱私優先）。這些 Stars 的興趣集中在 Agent 框架、程式碼搜尋、終端使用追蹤等工具鏈領域，與 OpenClaw 生態高度相關。
